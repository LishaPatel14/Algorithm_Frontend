Array Sum Challenge Completed

Find the sum of positive numbers in an array, ignoring negative values.

def sum_positive(arr):
    return sum(x for x in arr if x > 0)

print(sum_positive([1, -2, 3, 4, 5]))

Array Filter Pending

Find out negative numbers from an array and return new array.

def get_negatives(arr):
    return [x for x in arr if x < 0]

print(get_negatives([1, -2, 3, -4, 5]))

Maximum Value Completed

Find maximum positive number in an array.

def max_positive(arr):
    return max(x for x in arr if x > 0)

print(max_positive([1, -2, 3, 4, 5]))

Array Transform Pending

Transform array by squaring positive numbers.

def square_positives(arr):
    return [x**2 if x > 0 else x for x in arr]

print(square_positives([1, -2, 3, 4, 5]))
